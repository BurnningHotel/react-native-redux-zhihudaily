# react-native-redux-zhihudaily
使用react-native和redux重构的知乎日报，
本项目兼容ios版本，但界面只依照知乎日报Android端编写

## Getting started
```shell
npm install # install dependencies
npm run android # enjoy it
npm run ios # run ios version
```
